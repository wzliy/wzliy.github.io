如何设计一个最简网关

网关的主要原理就是拦截url请求，进行统一处理

我们可以先设置一个拦截器：继承OncePerRequestFilter

@Slf4j
public class ReverseProxyFilter extends OncePerRequestFilter {
    protected final RequestDataExtractor extractor;

    public ReverseProxyFilter(RequestDataExtractor extractor) {
        this.extractor = extractor;
    }
    @Override
    protected void doFilterInternal(HttpServletRequest request, HttpServletResponse response, FilterChain filterChain) throws ServletException, IOException {
        String originUri = extractor.extractUri(request);
        String originHost = extractor.extractHost(request);

        log.info("request uri : {}", originUri);
        log.info("request host : {}", originHost);
        response.setStatus(200);
        response.getOutputStream().write("success".getBytes());

    }
}

先进行验证：返回成功200状态码，以及success

将拦截器自动注入，这样SpringBoot容器启动的时候就会自动加载拦截器

@Configuration
public class FaradayConfiguration {
    @Bean
    @ConditionalOnMissingBean
    public RequestDataExtractor faradayRequestDataExtractor() {
        return new RequestDataExtractor();
    }

    @Bean
    @ConditionalOnMissingBean
    public ReverseProxyFilter faradayReverseProxyFilter(RequestDataExtractor extractor) {
        return new ReverseProxyFilter(extractor);
    }
}

这个是一个比较完整的网关的架构设计，我们参照这个来进行完善我们自己的网关服务。

路由映射表就是请求网关服务时的url映射到我们需求请求的微服务上，比如，我们请求网关服务的url为，http://localhost:80/account/create，我们可以通过配置account,让它映射到我们的account微服务。

首先，路由解析就是我们刚刚创建的拦截器，我们先在里面实现查询路由映射表的功能。

mapper ：对应路由映射表

MappingProvider是路由映射表实现的一个超类，它里面的方法，有两个是已经实现好的，一个是resolveMapping, 一个是updateMappings。resolveMapping就是你传一个主机头，它能帮你解析出来对应的路由映射，就是MappingProperties, 路由映射里面就是对应的路由名，主机头，目标服务地址等。updateMappings能够更新你的路由映射信息。

有两个具体的实现，一个是基于配置文件的ConfigurationMappingsProvider,一个是基于编程的方式的ProgrammaticMappingsProvider。

ConfigurationMappingsProvider：获取配置信息里的静态配置信息

ProgrammaticMappingsProvider：不需要动态更新，使用ServiceDirectory一次性构建好配置信息。

http: HttpClient映射表

ReverseProxyFilter：主流程请求响应处理，扩展了OncePerRequestFilter, 使用doFilterInternal处理请求

OncePerRequestFilter是Spring web内置的一个filter类，在Spring中，Filter默认继承OncePerRequestFilter
